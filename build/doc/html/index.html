About

World Imaker is a 3D scene editor-visualizer made with OpenGL/C++.

Installation

Tools required

  • OpenGL 3.0.0 or later: OpenGL Website
  • SDL 2.0: SDL Website

Running the program

  • Step 1: Clone the repo from Github.
  • Step 2: Create the build directory if doesn't exist. Then move into it.
  • Step 3: Run CMake and build the project.
  • Step 4: Execute the program in the bin folder.
